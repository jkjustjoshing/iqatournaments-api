Tournament Management API


This API is the backend of a tournament management system for Quidditch. It will keep track of team records, games, and rankings. 

In the future this will also include management for scorekeepers during a game and scheduling.


Tournament
GET   /tournaments
GET   /tournament/:id
PATCH /tournament/:id
GET   /tournaments/new
POST  /tournaments/new

Game
GET   /games
GET   /game/:id
PATCH /game/:id
GET   /games/new
POST  /games/new
